Key Features

Smooth scroll-based animations (GSAP / ScrollTrigger)

Cinematic section transitions

Responsive design for all screen sizes

Clean and minimal UI focused on visuals

Optimized structure for performance and scalability

Modular and maintainable front-end codebase

Purpose

This project serves as a professional portfolio showcase for photographers and filmmakers, as well as a demonstration of advanced front-end animation techniques and modern web design principles.
